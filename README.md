# TTX情報リテラシー教育ゲーム

## 概要

このプロジェクトは、情報操作に対する心理的抵抗力を育成するための教育ゲームです。**接種理論**（Inoculation Theory）に基づき、弱毒化された情報操作を体験することで、現実世界での誤情報に対する免疫を獲得することを目的としています。

## 研究背景

- **テーマ**: 接種理論を用いた情報リテラシー教育
- **手法**: タブルトップ演習（TTX）形式のインタラクティブゲーム
- **目標**: メタ認知的気づきを促進し、操作的レトリックへの抵抗力を高める

## ゲームの種類

### 1. 防御側TTX（`DefensiveTTXGame.jsx`）
プレイヤーは情報を**見抜く側**として、3人の情報操作者の主張に対応します。

**特徴**:
- 真偽が混在した情報
- 4つのアクション: 検証、質問、信じる、無視
- 限られたリソースでの戦略的判断
- 相互作用するキャラクター

### 2. 攻撃側シミュレーター（`ManipulatorSimulator.jsx`）
プレイヤーは情報**操作する側**として、市民を説得します。

**特徴**:
- 各市民に対して3ラウンドの段階的説得
- ラウンドごとに異なる段階の文章（導入→深化→クロージング）
- 市民の反応システム
- 説得度メーター
- 倫理的ジレンマの体験
- 代償の可視化

## 推奨プレイフロー

```
1. 防御側ゲーム（事前テスト）
   ↓
2. 攻撃側シミュレーター（接種体験）
   ↓
3. 防御側ゲーム（事後テスト）
   ↓
4. 効果測定
```

## セットアップ

### 前提条件
- Node.js (v14以上)
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/your-username/ttx-information-literacy-game.git
cd ttx-information-literacy-game

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm start
```

ブラウザで `http://localhost:3000` を開いてください。

## 教育的価値

### 接種理論の実装
1. **弱毒化された攻撃**: ゲーム内で安全に情報操作を体験
2. **能動的学習**: 自分で操作することで深く理解
3. **メタ認知の強化**: 「なぜ効果的か」を体験的に学習
4. **免疫の獲得**: 現実世界で遭遇した際の防御力向上

### 学べるレトリック手法
- ✅ 正直な説明
- 📊 統計の選択的提示
- 😢 感情への訴求
- 🎓 虚偽の権威づけ
- ⚠️ 恐怖の煽動
- 👥 社会的証明の捏造
- ⏰ 人工的緊急性

## 倫理的配慮

### ✅ このゲームは以下を目的としています
- 情報操作の手法を理解し、防御力を高める
- メタ認知的気づきを促進する
- 批判的思考力を育成する

### ⚠️ このゲームは以下を目的としていません
- 実際に人を騙す方法を教える
- 悪用を推奨する
- 誤情報の拡散

ゲームには必ず**警告文**と**デブリーフィング**が含まれており、学んだ手法の問題点と防御方法を明示しています。

## 研究での使用

このゲームは修士論文研究のために開発されました。研究や教育目的での使用を歓迎します。
